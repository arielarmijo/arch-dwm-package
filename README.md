# dwm
PKGBUILD with custom configuration and patches files for dwm-6.3.

![Desktop clean](screenshot-clean.png)
![Desktop dirty](screenshot-dirty.png)


# Install
```
git clone https://github.com/cochayuyo/dwm.git
cd dwm
makepkg -sci
```
