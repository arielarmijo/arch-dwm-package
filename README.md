# dwm
PKGBUILD with custom configuration and patches files for dwm-6.2.

![alt text](https://github.com/cochayuyo/dwm/blob/master/screenshot.png)


# Install
```
git clone https://github.com/cochayuyo/dwm.git
cd dwm
makepkg -sci
```
